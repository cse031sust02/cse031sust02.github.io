Understanding the Nginx Server Block

We can think of Server Blocks as specifications for individual web sites. Server blocks are the NGINX equivalent of Apache’s virtual hosts. We need to understand the directives and settings that make up the server block. Here is a sample virtual host : server { listen 80; server_name _; location / { root /usr/share/nginx/html; index index.html index.htm; } error_page 404 /404.html; location = /404.html { root /usr/share/nginx/html; } } We can see different types of directives such as listen, root, server_name, location etc. »
